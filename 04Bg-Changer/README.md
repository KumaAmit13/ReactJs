# Background Changer

## Description
This is a simple ReactJS project that allows users to change the background color of the page with a single click. It is a great beginner-friendly project to understand state management and event handling in React.

## Features
- Click a button to change the background color.
- Random color generation for dynamic background changes.
- Simple and clean UI.

## Technologies Used
- React.js
- JavaScript (ES6+)
- CSS

## Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/KumaAmit13/ReactJs.git
   ```
2. Navigate to the project directory:
   ```sh
   cd ReactJs/04Bg-Changer
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm start
   ```
5. Open your browser and go to `http://localhost:3000` to see the project in action.

## Usage
- Click the **Change Background** button to update the background color randomly.
- Experiment with different colors and refresh to reset.
