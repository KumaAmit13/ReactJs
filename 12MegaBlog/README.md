# MegaBlog - A Modern Blogging Platform

## 🚀 About the Project
MegaBlog is a modern, responsive blogging platform built using **React.js** and **Redux Toolkit**. It allows users to create, read, and manage blog posts with an intuitive UI. This project is part of the React.js learning journey and demonstrates concepts such as state management, routing, and API handling.

## 🛠 Tech Stack
- **Frontend:** React.js, Redux Toolkit, Tailwind CSS
- **State Management:** Redux Toolkit
- **Styling:** Tailwind CSS
- **Routing:** React Router
- **Real text editor** TinyMCE
- **Backend or Database** Appwrite

## 📦 Installation & Setup
To get started with MegaBlog, follow these steps:

### 1️⃣ Clone the repository:
```sh
git clone https://github.com/KumaAmit13/ReactJs.git
cd ReactJs/12MegaBlog
```

### 2️⃣ Install dependencies:
```sh
npm install
```

### 3️⃣ Run the development server:
```sh
npm start
```

The app will be available at **http://localhost:3000** and **https://megablogbyamit.netlify.app/**🚀
