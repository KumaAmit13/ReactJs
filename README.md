## Projects

This repository includes:

1. **01basicreact**: A foundational React project introducing core concepts.
2. **01vitereact**: An exploration of React development using Vite for enhanced performance.
3. **03tailwindprops**: Integration of Tailwind CSS in a React project to demonstrate styling with props.
4. **04Bg-Changer**: A background color changer application built with React.
5. **05passwordGenerator**: A React-based password generator with customizable options.
6. **06currencyConvetor**: A currency converter application developed in React.
7. **07reactrouter**: Demonstration of React Router for navigation within a React application.
8. **08minicontext**: A mini project showcasing React's Context API for state management.
9. **09themesswicher**: A theme switcher application allowing users to toggle between themes.
10. **10todocontextLocal**: A to-do list application utilizing React Context and local storage.
11. **12MegaBlog**: A comprehensive blogging platform built with React.
12. **Comments**: A comment section component developed in React.
13. **Counter**: A simple counter application to demonstrate React state management.
14. **Hooks**: Examples and explorations of various React hooks.
15. **ReactHookForm**: Integration of React Hook Form for form management.
16. **Router**: Additional demonstrations of routing in React applications.
17. **customReact**: Custom implementations and experiments within React.
18. **ownCuureny**: A personal take on a currency converter application.
19. **reduxToolkitTodo**: A to-do application utilizing Redux Toolkit for state management.
